# Welcome to Arig

Arig a desert planet with limited resources, a hot place with vast deserts where valuable resources can only be found underground or in the sand. Constant sandstorms make the use of robots unviable, and most of the surface is unsuitable for building without a proper foundation.

![Alt text](https://raw.githubusercontent.com/Syen-ce/planetaris-factorio-images/refs/heads/main/arig-gen-1.png)

# Is it safe to add the mod to an existing save?
Is completely safe to play in an existing save.

# Inside Arig

**(This section contains spoilers)**

### Cactus processing

Plant and harvest the natural cactuses on Arig's surface using the new condensing agricultural tower to obtain fruit and water.

![Alt text](https://raw.githubusercontent.com/Syen-ce/planetaris-factorio-images/refs/heads/main/arig-agriculture.png)

### Sifting and Compression

Use the sifter to sift large quantities of sand to extract basic ores and discover new sifting recipes.

Use the press to produce new materials and research to learn new application for compression.

![Alt text](https://raw.githubusercontent.com/Syen-ce/planetaris-factorio-images/refs/heads/main/arig-pressing-sifting.png)

### Rewards

-   The Press, with more efficient vanilla recipes
-   Bigger storage and logistical storage
-   Better solar panels
-   Over water/oil/sand solar panel and electric poles 
-   Condensing agricultural water, adds the ability to harvest fluid from plants using a new agricultural tower
-   New set of belts (Only if hyarion is not installed)
-   Water harvesting, produce water anywhere

### Compatibilities

The next mods have special compatibilities with Arig:

-   Krastorio 2 Space out, fully compatible (recipes and items)
-  Celestial Weather, custom atmosphere
-  AAI loaders, support for hyper belts
-  Any planet start
-  Visible planets in space
-  Crushing industry
-  Lignumis
-  Igrys
-  BzLead and BzTin
-  Canal excavator
-  Wood industry
